 - 팀장  : 이재우 - 로그인/회원가입, 메인 페이지, 쇼핑몰
 - 부팀장 : 이동재 - 관리자 페이지(상품, 회원, 주문 관리), 상품 등록 및 수정
 - 팀원1 : 김예슬 - 차트 페이지, 상품 리뷰/구매내역, 식단 페이지
 - 팀원2 : 유정현 - 컨설팅 질문/답변
 - 팀원3 : 정연재 - 기업 상세 페이지, 컨설팅 신청, 컨설팅 결제
